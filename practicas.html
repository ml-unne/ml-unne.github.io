      Uso de jupyter

      Hay dos alternativas, usar el google-colab, para esto hay que subir el archivo al google-drive.
      https://colab.research.google.com

      La otra opción es usar un servidor. Para usar jupyter en el servidor realizar:
      usuario@labif8:~$ ssh -L 8080:localhost:8080 "usuario"@10.40.60.207
      Una vez en la terminal del servidor hacer: 
      usuario@yakaira:~$ jupyter notebook --no-browser --port=8080
      Va a aparecer un mensaje que dice:
      To access the notebook, open this file in a browser:
      file:///home/XXXXX/.local/share/jupyter/runtime/nbserver-xxxx-open.html
      Or copy and paste one of these URLs:
      http://localhost:8080/?token=0e1b33c2eba504d5d787cdasfdsafe285bb44918dc9dsdfa

      copiar la linea que dice

      http://localhost:8080/?token=xxxxxx

      y pastearla en un navegador, firefox, chrome, etc.

      Para acceder desde afuera de la facultad es necesario ingresar primero al servidor publico 200.45.54.94: 
       ssh usuario@200.45.54.94
      Una vez que ingreso al servidor publico, debe ingresar al servidor de ML, 10.40.60.207, 
       ssh usuario@10.40.60.207
      Para usar jupyter desde afuera de la facultad.
      1. Me conecto al servidor público, y luego al servidor de ML:
      ssh -4fNL 2222:10.40.60.207:22 -4fNL 8080:10.40.60.207:8080 200.45.54.94
      1. Luego abro la conexion en la maquina local
      ssh -p 2222 localhost
      1. Abrir una jupyter notebook sin browser:
      jupyter notebook --no-browser --port=8080 --ip=0.0.0.0

      Copiar el token que aparece.
